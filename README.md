# Balanced Habits and Academic Performance 

## Project Overview
This project analyzes how study hours, sleep duration, and screen time together affect students' academic performance using Tableau.

## Visualizations
- Study Hours vs Grade Score (Scatter Plot)
- Sleep Hours vs Screen Time (Scatter Plot)
- Balanced vs Imbalanced Habits (Bar Chart)

## Key Insight
Balanced daily habits lead to better academic outcomes compared to extreme or imbalanced routines.
